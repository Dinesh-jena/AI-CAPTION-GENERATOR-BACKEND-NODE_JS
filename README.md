# 🚀 AI Caption Generator Backend (Node.js)

A scalable backend built with **Node.js**, **Express.js**, **MongoDB**, **Socket.IO**, and **Google Gemini AI** that generates AI-powered captions from uploaded images. The backend follows a clean architecture with authentication, cloud image storage, real-time communication, and AI integration.
